# student-credentials
#Overview
This project shows how to implement and application with PostgreSQL. This simple program performs the CRUD operations. CRUD meaning Create, Read , Update and Delete operations.

##DATABASE SETUP
Open pgAdmin
Created a database named studentdatabase
Open the query tool for studentdatabase
copy and execute the contents of students.sql

#Application 
The program requires Python 3, PostgreSQL and psycopg2 library installed

#How to run 
Open student.py 
type "python student.py" in terminal to execute
